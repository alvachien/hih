Main page of HIH

HIH, the abbreviation of Home Information Hub, which is built targets for all necessary information we need cross the home. The PoC here, demonstrates how to use jQuery Easy UI Library, and how to validate the form .
It also contains the objects validation, including:
(1) Reading data from MySql;
(2) Writing data back to MySql;
(3) Using jQuery to format the HTML output;
(4) Validate the framework may used;

This is the main page.
